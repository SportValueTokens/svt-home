New Digital Asset class

Sport Value Tokens represent athletes in different sports (Football, Basketball, Cricket, etc) offering a bet on the expected future performance of the athlete. Token holders will receive payouts proportional to in-game performance of each player. Our intention is to provide alternative, un-correlated to existing markets digital assets for sport fans as well as professional traders.

Sport Value Coin (SVC)

SVC is an utility token capped at 100 million. It will be used for different purposes:
- performance payouts

How It Works -
Tokenisation

An athlete will be represented by a crypto token, created by the platform

Tokens can be exchanged between holders independently

In practice, they need a dedicated exchange where they can trade these tokens

Each token holder will receive a payout based on the performance of the athlete for each game

Performance Payouts -
Crypto Betting Pool

Payouts will be funded by sale proceeds of new tokens. Tokens will be minted at regular intervals.

Payout amounts will be calculated using a predefined formula using externally provided, verifiable performance of the athlete

Traders may want to buy tokens for athletes that perform well, therefore driving price up or sell tokens of non performing athletes
